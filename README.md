# Wisnu Portfolio

## Upload ke GitHub Pages
1. Buat repository baru.
2. Upload seluruh isi folder ini.
3. Settings > Pages.
4. Source: Deploy from branch.
5. Branch: main / root.
6. Simpan dan tunggu deployment.

## Custom Domain
Tambahkan file CNAME berisi domain Anda lalu atur DNS di Exabytes.

Ganti:
- assets/ dengan foto Anda
- Contact
- Link LinkedIn
- Tombol Download CV
